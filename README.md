# 📦 About

This repository contains a collection of scripts and developer kits designed to quickly prepare Linux machines for development environments.

The goal is to automate repetitive setup tasks and provide a reusable toolkit for:

- ⚙️ Workstation provisioning
- 🐳 Docker environments
- 💻 Development tools
- ☕ Java setup
- 🟩 Node.js environments
- 🐍 Python environments
- 🔥 Dev productivity utilities
- 🧰 CLI tooling
- 🛠️ Future DevOps & Kubernetes kits

# ⚡ Quick Start

## Clone repository

```bash
git clone https://github.com/YOUR_USERNAME/linux-dev-setup.git
```

## Enter project folder

```bash
cd linux-dev-setup
```

## Give execution permission

```bash
chmod +x scripts/*.sh
```

## Run scripts

### Install developer tools

```bash
./scripts/softwares_dev.sh
```

### Install Docker

```bash
./scripts/docker_install.sh
```

---

# 🐳 Docker Notes

After Docker installation, reload your session or run:

```bash
newgrp docker
```

Test Docker:

```bash
docker run hello-world
```
